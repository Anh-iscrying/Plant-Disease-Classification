# 🌿 Plant Disease Classification using CNN

Hệ thống nhận diện bệnh hại trên lá cây sử dụng 3 kiến trúc mạng Deep Learning: MobileNetV2, ResNet50 và EfficientNet-B0.

## 📊 Kết quả thực nghiệm (20 Epochs)
| Model | Accuracy | F1-Score |
| :--- | :---: | :---: |
| **MobileNetV2** | 99.55% | 99.55% |
| **ResNet50** | 99.45% | 99.45% |
| **EfficientNet-B0** | 99.55% | 99.55% |

## 🚀 Tính năng
- Nhận diện 15 loại bệnh trên Cà chua, Khoai tây, Ớt chuông.
- So sánh kết quả trực diện giữa 3 mô hình.
- Giao diện web trực quan bằng Streamlit.

## 🛠 Hướng dẫn chạy App
1. Clone dự án: `git clone https://github.com/your-username/your-repo-name.git`
2. Cài đặt thư viện: `pip install -r requirements.txt`
3. Chạy App: `streamlit run app.py`
